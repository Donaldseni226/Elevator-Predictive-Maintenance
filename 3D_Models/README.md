# 3D Models for Sensor Mounts

This directory contains 3D model files that are designed for sensor mounts used in the Elevator Predictive Maintenance system. 

## Available Models
- **Mount for Temperature Sensor**: A model designed to securely hold a temperature sensor in place.
- **Mount for Vibration Sensor**: This model provides stability and ensures accurate readings from the vibration sensor.
- **Mount for Humidity Sensor**: A customizable mount that fits various humidity sensor models.

## Printing Instructions
- Recommended material: PLA or ABS plastic.
- Layer height: 0.2mm for optimal detail.
- Ensure supports are enabled for parts with overhangs.

Feel free to modify the models as needed to better suit your individual requirements.
